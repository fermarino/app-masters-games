# Projeto Frontend React - Catálogo de Games

Projeto de catálogo de jogos desenvolvido com Next.js, que permite aos usuários obter informações sobre jogos a partir de uma API externa.

## Stacks utilizadas

- React
- Next.js
- CSS Modules
- Node.js
- Git
- Firebase
